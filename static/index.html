Welcome to PATP

How would you like to use PTAP?

O mode API Proxy

In this mode, PTAP will overwrite all request received, then sends them to Twitter.
You need to have an UID to use O mode.

Register your UID HERE

T mode API Proxy

In this mode, PTAP act as a simple proxy, forwarding requests to Twitter.
You need to set the REST base and oAuth base url to the following:

REST:
/api/t/1.1/

oAuth:
/api/t/oauth/
